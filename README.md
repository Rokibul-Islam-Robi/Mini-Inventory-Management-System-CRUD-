# Mini Inventory Management System

Technology:
- Core PHP
- MySQL
- XAMPP / Apache
- PDO database connection
- Simple company-standard folder separation

## Installation Steps

1. Install XAMPP.
2. Start Apache and MySQL.
3. Copy the folder `mini_inventory_management_system` into:
   `C:\xampp\htdocs\`
4. Open phpMyAdmin.
5. Import:
   `database/inventory_db.sql`
6. Run:
   `http://localhost/mini_inventory_management_system/`
7. Login:
   - Username: `admin`
   - Password: `password`

## Main Features

- Login
- Product category CRUD
- Product entry CRUD
- Stock in
- Stock out with stock validation
- Current stock report
- Date-wise report
- Print option
